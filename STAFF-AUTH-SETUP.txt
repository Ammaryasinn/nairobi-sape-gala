PART 1: ENABLE FIREBASE AUTHENTICATION
───────────────────────────────────────────────────────────────

STEP 1: Go to Firebase Console
☐ Open: https://console.firebase.google.com
☐ Select your project: sape-gala-2026

STEP 2: Enable Authentication
☐ Click "Build" in left sidebar
☐ Click "Authentication"
☐ Click "Get started" button

STEP 3: Enable Email/Password Sign-In
☐ Click the "Sign-in method" tab at the top
☐ Find "Email/Password" in the list
☐ Click on "Email/Password"
☐ Toggle ON the "Enable" switch
☐ Leave "Email link (passwordless sign-in)" OFF
☐ Click "Save"

✅ Authentication is now enabled!

TROUBLESHOOTING
───────────────────────────────────────────────────────────────

PROBLEM: "Invalid email or password" error
SOLUTION:
☐ Check email is EXACTLY as entered in Firebase Console
☐ Check password (case-sensitive, no extra spaces)
☐ Verify account exists in Firebase → Authentication → Users
☐ Try password reset if needed

PROBLEM: Login works but redirects back to login page
SOLUTION:
☐ Check browser console (F12) for errors
☐ Verify Firebase Auth SDK loaded correctly
☐ Check firebase-config.js has correct credentials
☐ Clear browser cache and try again

PROBLEM: Can't access staff-checkin.html even when logged in
SOLUTION:
☐ Check browser console for JavaScript errors
☐ Verify staff-auth.js is loaded (check Network tab)
☐ Try logging out and logging in again
☐ Check Firestore rules are published

PROBLEM: Customers can't buy tickets
SOLUTION:
☐ Check the backend function is deployed successfully
☐ Check Firestore rules were deployed from firestore.rules
☐ Verify firebase-config.js is correct
☐ Check browser console for errors

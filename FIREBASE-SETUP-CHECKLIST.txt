🔥 FIREBASE SETUP STEPS
───────────────────────

STEP 1: CREATE FIREBASE PROJECT (~10 min)
☐ Go to https://console.firebase.google.com
☐ Sign in with Google account
☐ Click "Add project" or "Create a project"
☐ Project name: sape-gala-2026
☐ Enable Google Analytics (recommended)
☐ Wait for project creation
☐ Click "Continue" when ready

STEP 2: ENABLE FIRESTORE DATABASE (~5 min)
☐ Click "Build" → "Firestore Database"
☐ Click "Create database"
☐ Select "Start in production mode"
☐ Choose location: eur3 (europe-west) or asia-south1 (Mumbai)
☐ Click "Enable"
☐ Wait for database creation (1-2 minutes)

STEP 3: CONFIGURE SECURITY RULES (~3 min)
☐ Go to Firestore Database → "Rules" tab
☐ Copy the rules from FIREBASE-SETUP-GUIDE.md
☐ Paste into the rules editor
☐ Click "Publish"
☐ Confirm by clicking "Publish" again

STEP 4: GET CONFIGURATION KEYS (~5 min)
☐ Click home icon (Firebase Console main page)
☐ Click "</>" icon (Web app)
☐ App nickname: SAPE Gala Website
☐ Optional: Check "Also set up Firebase Hosting"
☐ Click "Register app"
☐ COPY these 6 values:

    ✏️ apiKey:             _________________________________
    
    ✏️ authDomain:         _________________________________
    
    ✏️ projectId:          _________________________________
    
    ✏️ storageBucket:      _________________________________
    
    ✏️ messagingSenderId:  _________________________________
    
    ✏️ appId:              _________________________________

☐ Click "Continue to console"

STEP 5: UPDATE YOUR WEBSITE CODE (~2 min)
☐ Open: C:\Users\kyalo\nairobi-sape-gala\firebase-config.js
☐ Find lines 5-11 (the firebaseConfig object)
☐ Replace "YOUR_API_KEY_HERE" with your actual apiKey
☐ Replace "YOUR_PROJECT_ID" with your actual projectId (3 times)
☐ Replace "YOUR_MESSAGING_SENDER_ID" with your actual ID
☐ Replace "YOUR_APP_ID" with your actual appId
☐ Save file (Ctrl + S)

STEP 6: TEST THE CONNECTION (~3 min)
☐ Open index.html in browser
☐ Press F12 to open Developer Console
☐ Look for: "✅ Firebase initialized successfully"
☐ Click "Get Your Ticket"
☐ Fill test data and submit
☐ Check ticket appears in modal
☐ Go to Firebase Console → Firestore Database
☐ Verify "tickets" collection exists
☐ Verify test ticket data appears

🆘 TROUBLESHOOTING
──────────────────

PROBLEM: "Firebase not configured" error
SOLUTION:
☐ Check all 6 config values are replaced
☐ No extra spaces or quotes
☐ apiKey doesn't say "YOUR_API_KEY_HERE"
☐ Clear browser cache (Ctrl + Shift + Delete)

PROBLEM: "Permission denied" error
SOLUTION:
☐ Check Firestore Rules tab
☐ Ensure rules allow "allow read, write: if true;"
☐ Click "Publish" button
☐ Wait 30 seconds and try again

PROBLEM: Data not showing in Firestore
SOLUTION:
☐ Check browser console for errors (F12)
☐ Verify correct Firebase project selected
☐ Try creating another test ticket
☐ Check Network tab for failed requests

PROBLEM: Old localStorage data showing
SOLUTION:
☐ Open browser console (F12)
☐ Type: localStorage.clear()
☐ Press Enter
☐ Refresh page (Ctrl + R)
